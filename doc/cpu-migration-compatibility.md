## Configuration Version

### Configuration Version 7.0 (Baseline)

**Integer / General**: CMPXCHG16B, POPCNT, LZCNT, MOVBE, BMI1, BMI2, ADX, INVPCID, PCID, PREFETCHW, FSGSBASE, ERMS, RDRAND, RDSEED, LAHF/SAHF, XSAVE-X87

**Vector / FP**: SSE3, SSSE3, SSE4.1, SSE4.2, AVX, AVX2, FMA, F16C, XSAVE, XSAVEOPT

**Crypto**: AES, PCLMULQDQ

**AMD-only**: SSE4A, XOP, 3DNOW, 3DNOWEXT, MMXEXT, MISALIGNSSE, FMA4, ALTMOVCR8

**Transactional Memory**: HLE, RTM

**Security / Mitigation**: SMEP, SMAP, IBRS, STIBP, IBPB, SSBD, MD_CLEAR, TSX_CTRL, NPIEP1
