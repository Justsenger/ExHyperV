# Privacy Policy for ExHyperV / 隐私政策

**Last updated / 生效日期**: 2026-07-24

ExHyperV is a free and open-source graphical Hyper-V management tool. We respect your privacy and are committed to protecting it.

ExHyperV 是一款开源免费的图形化 Hyper-V 管理工具。我们高度尊重并致力于保护您的个人隐私。

---

### 1. Information Collection / 数据收集
- **No Personal Data Collected**: ExHyperV does NOT collect, store, transmit, or share any personally identifiable information (PII), usage telemetry, or sensitive system data.
- **不收集任何个人数据**：ExHyperV 不会收集、存储、上传或共享任何用户的个人身份信息、使用痕迹或敏感系统数据。

### 2. Local Execution / 本地化处理
- All operations, WMI/HCS API calls, and virtual machine configurations are executed strictly on your local computer.
- 所有软件操作、WMI/HCS 接口调用及虚拟机配置均严格在您的本地计算机上完成。

### 3. Network Access / 网络访问
- The application only accesses the network when checking for software updates from GitHub Releases, or when you explicitly configure network-related VM settings.
- 本软件仅在向 GitHub Releases 检查软件版本更新，或您主动配置虚拟机网络功能时才会使用网络。

### 4. Contact Us / 联系方式
If you have any questions regarding this Privacy Policy, please feel free to open an Issue on GitHub:
如有任何隐私相关疑问，请通过 GitHub Issues 联系我们：
https://github.com/Justsenger/ExHyperV/issues
